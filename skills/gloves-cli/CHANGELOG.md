# gloves-cli

## 0.1.0

### Minor Changes

- Initial release of the `gloves-cli` skill for operating the `gloves` secrets manager.
- Added task playbooks for agent secret lifecycle, human request lifecycle, and expiry verification.
- Added guardrails to keep raw secret values out of agent memory and notes.
- Added command reference covering `init`, `set`, `get`, `env`, `request`, `approve`, `deny`, `list`, `revoke`, `status`, and `verify`.
